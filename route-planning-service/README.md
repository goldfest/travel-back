# Route Planning Service

Микросервис для управления маршрутами путешествий, оптимизации и планирования.

## Основные функции

### Управление маршрутами (UC-13, UC-15, UC-16, UC-17, UC-18)
- Создание, чтение, обновление, удаление маршрутов
- Архивирование/разархивирование маршрутов
- Дублирование маршрутов

### Планирование точек маршрута (UC-14)
- Добавление/удаление точек (POI) в маршруты
- Изменение порядка точек
- Распределение по дням

### Оптимизация маршрутов (UC-20, UC-26, UC-27)
- Автоматическая оптимизация по времени, расстоянию, рейтингу
- Учет часов работы объектов
- Разные режимы транспорта (пешком, транспорт, авто)

### Оффлайн работа (UC-19)
- Подготовка данных для оффлайн использования
- Синхронизация оффлайн данных
- Управление оффлайн хранилищем

### Экспорт и шаринг
- Экспорт в PDF, GPX, JSON форматы
- Генерация шаринговых ссылок

### Планирование по дням (UC-28)
- Создание структуры маршрута по дням
- Планирование времени на каждый день

## Технологии

- Java 21
- Spring Boot 3.2.0
- PostgreSQL с PostGIS
- Redis для кэширования
- Hibernate/JPA
- MapStruct для маппинга
- OpenFeign для межсервисного взаимодействия
- Swagger/OpenAPI для документации

## Запуск сервиса

### Требования
- Java 21+
- Maven 3.8+
- Docker и Docker Compose (для запуска в контейнерах)

### Локальный запуск (development)
```bash
cd route-planning-service
mvn spring-boot:run -Dspring-boot.run.profiles=dev
