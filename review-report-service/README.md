# Review & Reports Service

Микросервис для управления отзывами и жалобами в приложении для путешественников.

## Функциональность

### Управление отзывами:
- Создание, чтение, обновление и удаление отзывов
- Лайки/дизлайки для отзывов
- Модерация отзывов (скрытие/показ)
- Статистика отзывов по POI

### Управление жалобами:
- Создание жалоб на отзывы и POI
- Модерация жалоб (обработка, изменение статуса)
- Отслеживание статусов жалоб
- Уведомления о новых жалобах

## Технологии

- Java 21
- Spring Boot 3.2.0
- Spring Data JPA
- PostgreSQL
- Redis (кэширование)
- OpenFeign (межсервисная коммуникация)
- Swagger/OpenAPI 3
- JWT аутентификация
